# 2 Seasons — Next.js Web App

Scroll-expansion hero home page with React, Next.js, Tailwind, and Framer Motion.

## Setup

```bash
cd web
npm install
```

## Run

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```

## Structure

- `app/page.tsx` — Home page with scroll-expansion hero
- `components/ui/scroll-expansion-hero.tsx` — Scroll-to-expand media component
- `public/100plots/` — Video and images (copied from parent)
- `public/assets/` — Site images (copied from parent)

## Dependencies

- Next.js 15, React 19
- Tailwind CSS
- Framer Motion
- TypeScript
